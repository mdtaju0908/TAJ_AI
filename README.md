# 🤖 TAJ AI

TAJ AI is a modern AI chat assistant built with **Next.js + Node.js + Gemini API**.  
It provides a clean Glassmorphism UI and powerful AI features like Thinking Mode, Deep Research, and Web Search.

---

## 🚀 Features

- 💬 AI Chat Interface
- 🤖 Gemini AI Integration
- ⚡ Thinking Mode
- 🔍 Deep Research Mode
- 🌐 Web Search Mode
- 📋 Copy Messages
- ✏️ Edit Messages
- 📥 Download Responses
- 🌙 Dark Glassmorphism UI
- 📱 Responsive Design

---

## 🛠 Tech Stack

### Frontend
- Next.js
- React
- TailwindCSS
- Lucide Icons

### Backend
- Node.js
- Express.js
- Gemini API
