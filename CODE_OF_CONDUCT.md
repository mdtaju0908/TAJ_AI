# Code of Conduct

## TAJ AI Community Guidelines

Thank you for being part of the **TAJ AI** community.
To maintain a welcoming and professional environment, all participants are expected to follow this Code of Conduct.

---

## Our Standards

Examples of behavior that contribute to a positive environment include:

* Being respectful and professional in communication
* Providing constructive feedback
* Helping other contributors learn and grow
* Focusing on improving the project

Examples of unacceptable behavior include:

* Harassment or discrimination of any kind
* Offensive or abusive language
* Spamming or trolling
* Publishing private information without permission
* Any activity that disrupts the project or community

---

## Scope

This Code of Conduct applies to all interactions within the **TAJ AI repository**, including:

* GitHub Issues
* Pull Requests
* Discussions
* Project collaboration
