# Contributing to TAJ AI

Thank you for your interest in contributing to **TAJ AI**.

Please read these guidelines before contributing to ensure a smooth collaboration process.

---

## Contribution Policy

⚠️ **Important Notice**

The TAJ AI project is currently provided for **review and learning purposes only**.

Cloning, copying, modifying, or using the source code in other projects without permission is **not allowed** according to the project license.

However, you may still contribute by:

* Reporting bugs
* Suggesting improvements
* Sharing feature ideas
* Helping improve documentation

---

## How to Contribute

### 1. Report a Bug

If you find a bug:

* Check if the issue already exists
* Open a new GitHub issue
* Provide detailed steps to reproduce the problem
* Include screenshots if possible

---

### 2. Suggest a Feature

If you want to suggest a new feature:

* Open a **Feature Request** issue
* Clearly explain the idea
* Describe how it improves the project

---

### 3. Improve Documentation

You can help by improving:

* README
* Guides
* Code comments
* Documentation files

---

## Guidelines

When interacting with the project:

* Be respectful and professional
* Follow the repository Code of Conduct
* Provide clear and useful information in issues
